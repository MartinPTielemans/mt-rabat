# Task Completion Checklist

When a task is completed, ensure the following:

## Before Committing
1. Run `bun run lint` to check for linting errors
2. Run `bun run build` to verify static generation works
3. Test changes locally with `bun run dev`

## Commit Guidelines
- No emojis in commit messages
- Keep commits simple and descriptive
- No AI attribution in commits
- No blank lines with whitespace

## Verification
- Ensure TypeScript compilation succeeds
- Check that all content is properly typed
- Verify images are in `public/images/`
