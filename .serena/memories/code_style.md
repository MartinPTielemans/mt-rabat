# Code Style and Conventions

## General
- Use UK English in comments and documentation
- No emojis in commit messages
- Keep commits simple, no AI attribution

## TypeScript
- Use TypeScript throughout with strict typing
- All content is strongly typed with TypeScript interfaces
- Content files in `app/data/` directory

## React/Next.js
- Use React 19 features
- Use Next.js 15 App Router
- Static generation preferred

## CSS
- Tailwind CSS v4 for styling
- Custom design system in tailwind.config.ts
- Global styles in app/globals.css

## Principles
- DRY (Don't Repeat Yourself)
- SOLID principles
- KISS (Keep It Simple, Stupid)
